# js-lib-example

A simple JavaScript library.

## Installation

```bash
npm install js-lib-example
```

## Usage

```javascript
const lib = require('js-lib-example');

// Add two numbers
const sum = lib.add(2, 3); // 5

// Subtract two numbers
const difference = lib.subtract(5, 2); // 3
```
